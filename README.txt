WHAT CAN YOU DO IN THIS :-                                                                                                       
                                                                                                                                                            
(1)   	Enter your desired hostname, username and password 
(2)   	see created databases
(3)   	use database
(4)   	delete a database
(5)   	create table with n number of columns and desired constrains
(6)   	show created tables
(7)   	add an attribute in a table
(8)   	delete an attribute in a table
(9)   	modify attribute definition
(10) 	input n no of records in a table
(11) 	see table structure
(12) 	print whole table 
(13) 	print specific column records
(14) 	conditional printing of a full record based on specified column and a value in a column
(15) 	an interactive and clean menu
(16) 	work on databases as much you need
(17) 	all this comes along with power of CLI i.e minimum performance issues
